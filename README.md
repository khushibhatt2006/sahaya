# Sahaya - Kindness Connecting Platform ❤️

A beautiful donation platform for books, food, medicine, clothes & utensils.

**Features:**
- Warm, cute UI matching the logo (brown + pink heart + cream)
- Three roles: Donor, Receiver, Admin
- Simple matching system
- Impact dashboard for admin
- Enhanced cards with hover effects

## How to Run
1. `pip install -r requirements.txt`
2. `streamlit run main.py`

**Default Logins:**
- Admin: `admin` / `admin123`
- Donor: `donor1` / `123`
- Receiver: `receiver1` / `123`
